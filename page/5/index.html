java 数字格式化

Posted on 2017-03-19 Edited on 2021-09-14

前言

数字的格式化 是工作中不可避免要接触到的问题，我们经常碰到这些的需求：

  1. 金额以 ”¥999,999.99“ 格式 (带金额符，逗号分组，小数点保留两位)

  2. 数字以 ”999,999.99“格式

  3. 百分比 ”％“ 或 千分比 ”‰“ 例：0.45 —> 45%

  4. 小数点后最多保留两位小数，如果有零，则直接省略 例：3.141 —> 3.14

    ​ 3.100 —> 3.1

    ​ 3.000 —> 3

  5. 精度模式 ”舍“ ”入“ 或 ”四舍五入“

如此，就得好好整理一下。
