Android Studio 常见问题

发表于 2018-06-16 | 分类于 工具 | 阅读次数:

最近，Android Studio 更新到后，遇到了一些问题，记录一下。


Android Studio 配置

Help -> Edit Custom VM Option -> studio.vmoptions文件

根据配置，自行调整。保存后，需要重启生效。

APP 无法运行

升级了 Android Studio 后，APP 就无法启动，显示红叉。

这种情况一般需要检查 Android SDK，需要下载更新版本。

Preferences -> 搜索 Android SDK -> Edit -> Next-> Finish

重新编译，就可以了。


依赖下载失败

  1. 在 As Build 面板中，查看依赖库的下载地址，是否能使用浏览器下载，如果不能下载，原因很多，可能是远程服务器无法访问，可能给被运营商拦截，可能设置了全局代理，可能网线断了。
  2. 查看 AS Prefereces -> HTTP Proxy 是否配置代理。
  3. 检查 gradle.properties 文件是否配置代理，一般这个很难察觉到，排查了各种原因，折腾了许久后才发现，此文件下居然配置了代理，cry…..


无法预览布局

编辑 xml 时，无法正常预览布局，布局不正常。

查看报错信息，解决问题，如无问题，则尝试切换主题。

还可以切换设备，比如默认设备是 nexus 4 ，切换到 Nexus 5。
