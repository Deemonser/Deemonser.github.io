BigDecimal

еҸ‘иЎЁдәҺ 2017-11-14 | йҳ…иҜ»ж¬Ўж•°:

еңЁе•Ҷдёҡи®Ўз®—дёӯпјҢдёҖиҲ¬йғҪдјҡз”ЁеҲ° Java дёӯзҡ„ BigDecimal жҲ–иҖ… BigIntegerгҖӮ

иҝҷзҜҮж–Үз« е°ұжқҘи®°еҪ• BigDecimal зҡ„дёҖдәӣз”Ёжі•зҹҘиҜҶгҖӮ


Java еҹәжң¬ж•°жҚ®зұ»еһӢ

еңЁJAVAдёӯдёҖе…ұжңүе…«з§Қеҹәжң¬ж•°жҚ®зұ»еһӢпјҢд»–д»¬еҲҶеҲ«жҳҜ пјҡ
byteгҖҒshortгҖҒintгҖҒlongгҖҒfloatгҖҒdoubleгҖҒcharгҖҒboolean


ж•ҙж•°еһӢпјҡ

еңЁи®Ўз®—жңәдёӯпјҢжүҖжңүзҡ„ж–Үд»¶йғҪжҳҜд»ҘдәҢиҝӣеҲ¶зҡ„еҪўејҸеӯҳеӮЁзҡ„пјҢжүҖжңүи§„е®ҡпјҢдёҖдёӘеӯ—иҠӮпјҲbyteпјүеҚ 8дҪҚпјҲbitпјүдәҢиҝӣеҲ¶дҪҚж•°гҖӮеңЁиЎЁзӨәж•ҙж•°еһӢж—¶пјҢжңҖй«ҳдҪҚз¬ҰеҸ·дҪҚиЎЁзӨәж•°еӯ—зҡ„жӯЈиҙҹпјҢеү©дҪҷзҡ„дҪҚж•°иЎЁзӨәдёҖдёӘж•°еҖјпјҢе®ғд»¬еҗҲиө·жқҘиЎЁзӨәдёҖдёӘж•ҙж•°гҖӮ

  • byte пјҡеҚ 1дёӘеӯ—иҠӮпјҢз”Ё8дҪҚдәҢиҝӣеҲ¶иЎЁзӨәпјҢжүҖд»Ҙе…¶еҸ–еҖјиҢғеӣҙ -128~127пјҲ-2зҡ„7ж¬Ўж–№еҲ°2зҡ„7ж¬Ўж–№-1пјү
  • shortпјҡеҚ 2дёӘеӯ—иҠӮпјҢз”Ё16дҪҚдәҢиҝӣеҲ¶иЎЁзӨәпјҢжүҖд»Ҙе…¶еҸ–еҖјиҢғеӣҙ-32768~32767пјҲ-2зҡ„15ж¬Ўж–№еҲ°2зҡ„15ж¬Ўж–№-1пјү
  • intпјҡеҚ 4дёӘеӯ—иҠӮпјҢз”Ё32дҪҚдәҢиҝӣеҲ¶иЎЁзӨәпјҢе…¶еҸ–еҖјиҢғеӣҙ-2147483648~2147483647пјҲ-2зҡ„31ж¬Ўж–№еҲ°2зҡ„31ж¬Ўж–№-1пјү
  • longпјҡеҚ 8дёӘеӯ—иҠӮпјҢз”Ё64дҪҚдәҢиҝӣеҲ¶иЎЁзӨәпјҢе…¶еҸ–еҖјиҢғеӣҙ-9223372036854774808~9223372036854774807пјҲ-2зҡ„63ж¬Ўж–№еҲ°2зҡ„63ж¬Ўж–№-1пјү


жө®зӮ№еһӢпјҡ

жө®зӮ№еһӢзҡ„еӯҳеӮЁж–№ејҸдёҺж•ҙж•°еһӢдёҚеҗҢпјҢе…¶йҮҮз”Ёз§‘еӯҰи®Ўж•°ж–№ејҸиЎЁзӨәпјҢжңҖй«ҳдҪҚдҫқз„¶жҳҜиЎЁзӨәжӯЈиҙҹзҡ„з¬ҰеҸ·дҪҚпјҢз„¶еҗҺдёӯй—ҙеӨҡдәҶиЎЁзӨәжҢҮж•°зҡ„жҢҮж•°дҪҚпјҢе…¶дҪҷзҡ„жҳҜе°ҫж•°дҪҚгҖӮжө®зӮ№еһӢзҡ„иҢғеӣҙдё»иҰҒеҸ–еҶідәҺжҢҮж•°дҪҚпјҢзІҫеәҰеҸ–еҶідәҺе°ҫж•°дҪҚгҖӮjava дёӯй»ҳи®Өзҡ„жө®зӮ№еһӢжҳҜ doubleгҖӮ

  • floatпјҡеҚ 4дёӘеӯ—иҠӮпјҢз”Ё32дҪҚдәҢиҝӣеҲ¶иЎЁзӨәпјҢз¬¬1дҪҚз¬ҰеҸ·дҪҚиЎЁзӨәжӯЈиҙҹпјҢз¬¬2~9дҪҚиҝҷе…«дҪҚиЎЁзӨәжҢҮж•°дҪҚпјҢеҗҺйқў23дҪҚйғҪиЎЁзӨәе°ҫж•°гҖӮfloatзҡ„жҢҮж•°йғЁеҲҶжңү8bit(2^8)пјҢеҜ№еә”зҡ„жҢҮж•°иҢғеӣҙ-128~128пјҢе…¶еҸ–еҖјиҢғеӣҙ -2^128еҲ°2^128 пјҢз”Ёз§‘еӯҰи®Ўж•°иЎЁзӨә3.402823e+38 ~ 1.401298e-45пјҲe+38иЎЁзӨәжҳҜд№ҳд»Ҙ10зҡ„38ж¬Ўж–№пјҢеҗҢж ·пјҢe-45иЎЁзӨәд№ҳд»Ҙ10зҡ„иҙҹ45ж¬Ўж–№пјүпјҢfloatзҡ„е°ҫж•°дҪҚжҳҜ23bitпјҢеҜ№еә” 2^23=8388608пјҢжңҖеӨҡ7дҪҚжңүж•Ҳж•°еӯ—пјҢдҝқиҜҒжңү6дҪҚжңүж•Ҳж•°еӯ—гҖӮ
  • doubleпјҡеҚ 8дёӘеӯ—иҠӮпјҢз”Ё64дҪҚдәҢиҝӣеҲ¶иЎЁзӨәпјҢ1дҪҚз¬ҰеҸ·дҪҚпјҢз„¶еҗҺдёӯй—ҙ11дҪҚиЎЁзӨәжҢҮж•°дҪҚпјҢ52дҪҚе°ҫж•°гҖӮdoubleзҡ„жҢҮж•°йғЁеҲҶжңү11bit(2^11)пјҢеҜ№еә”зҡ„жҢҮж•°иҢғеӣҙ-1024~1024пјҢе…¶еҸ–еҖјиҢғеӣҙ -2^1024еҲ°2^1024 пјҢз”Ёз§‘еӯҰи®Ўж•°иЎЁзӨә1.797693e+308~ 4.9000000e-324пјҢdoubleзҡ„е°ҫж•°дҪҚжҳҜ52bitпјҢеҜ№еә”2^52 = 4503599627370496пјҢжңҖеӨҡ16дҪҚжңүж•Ҳж•°еӯ—пјҢдҝқиҜҒ15дҪҚжңүж•ҲгҖӮ


char еһӢ

ж–Үжң¬еһӢз”ЁдәҺеӯҳж”ҫеӯ—з¬Ұзҡ„ж•°жҚ®зұ»еһӢпјҢеҚ з”Ё2дёӘеӯ—иҠӮпјҢйҮҮз”Ёunicodeзј–з ҒпјҢдёҚиҝҮ8дҪҚзҡ„ASCIIз ҒеҢ…еҗ«еңЁUnicodeдёӯпјҢжҳҜд»Һ0~127е…је®№зҡ„гҖӮ

  • charпјҡеҚ 2дёӘеӯ—иҠӮпјҢз”Ё16дҪҚиЎЁзӨә


еёғе°”еһӢпјҡ

еёғе°”ж•°жҚ®зұ»еһӢеҸӘжңүдёӨдёӘеҸҜиғҪзҡ„еҖјпјҡtrueе’Ңfalse

  • booleanпјҡеёғе°”ж•°жҚ®зұ»еһӢжІЎжңүз»ҷеҮәе…·дҪ“зҡ„еҚ з”Ёеӯ—иҠӮж•°пјҢеӣ дёәеҜ№иҷҡжӢҹжңәжқҘиҜҙж №жң¬е°ұдёҚеӯҳеңЁ boolean иҝҷдёӘзұ»еһӢпјҢbooleanзұ»еһӢеңЁзј–иҜ‘еҗҺдјҡдҪҝз”Ёе…¶д»–ж•°жҚ®зұ»еһӢжқҘиЎЁзӨәгҖӮе®ҳж–№ж–ҮжЎЈе№¶жІЎжңүз»ҷеҮәзІҫзЎ®зҡ„е®ҡд№үпјҢгҖҠJavaиҷҡжӢҹжңәи§„иҢғгҖӢз»ҷеҮәдәҶ4дёӘеӯ—иҠӮпјҢе’Ңbooleanж•°з»„1дёӘеӯ—иҠӮзҡ„е®ҡд№үпјҢе…·дҪ“иҝҳиҰҒзңӢиҷҡжӢҹжңәе®һзҺ°жҳҜеҗҰжҢүз…§и§„иҢғжқҘпјҢжүҖд»Ҙ1дёӘеӯ—иҠӮгҖҒ4дёӘеӯ—иҠӮйғҪжҳҜжңүеҸҜиғҪзҡ„гҖӮ


еҪ“з„¶пјҢиҝҷзҜҮж–Үз« е№¶дёҚжҳҜи®ІдёҠйқўзҡ„еҹәжң¬ж•°жҚ®зұ»еһӢзҡ„гҖӮ


BigDecimal д»Ӣз»Қ

еңЁе®һйҷ…ејҖеҸ‘дёӯпјҢjava зҡ„жө®зӮ№еһӢеҸӘиғҪз”ЁжқҘиҝӣиЎҢз§‘еӯҰи®Ўз®—жҲ–е·ҘзЁӢи®Ўз®—пјҢеңЁеӨ§еӨҡж•°зҡ„е•Ҷдёҡи®Ўз®—дёӯпјҢдёҖиҲ¬йҮҮз”Ё java.math.BigDecimal зұ»жқҘиҝӣиЎҢзІҫзЎ®и®Ўз®—гҖӮе•Ҷдёҡи®Ўз®—дёӯпјҢиҰҒе°ҪеҸҜиғҪдҝқиҜҒиҝҗз®—зҡ„еҮҶзЎ®жҖ§пјҢжүҖд»ҘеңЁи®Ўз®—ж—¶иҰҒе°ҪеҸҜиғҪеӨҡзҡ„дҝқиҜҒжңүж•ҲдҪҚж•°пјҢеҮҸе°ҸзІҫеәҰжҚҹеӨұпјҢиҖҢ double жңҖеӨҡеҸӘиғҪдҝқиҜҒ16дҪҚжңүж•Ҳж•°еӯ—пјҢеӣ жӯӨпјҢJavaеңЁjava.mathеҢ…дёӯжҸҗдҫӣзҡ„APIзұ»BigDecimalпјҢз”ЁжқҘеҜ№и¶…иҝҮ16дҪҚжңүж•ҲдҪҚзҡ„ж•°иҝӣиЎҢзІҫзЎ®зҡ„иҝҗз®—гҖӮ


BigDecimal жҳҜдёҚеҸҜеҸҳзҡ„гҖҒд»»ж„ҸзІҫеәҰзҡ„жңүз¬ҰеҸ·еҚҒиҝӣеҲ¶ж•°пјҢз”ұд»»ж„ҸзІҫеәҰзҡ„ж•ҙж•°йқһж ҮеәҰеҖј е’Ң32 дҪҚзҡ„ж•ҙж•°ж ҮеәҰ (scale) з»„жҲҗгҖӮеҰӮжһңдёәйӣ¶жҲ–жӯЈж•°пјҢеҲҷж ҮеәҰжҳҜе°Ҹж•°зӮ№еҗҺзҡ„дҪҚж•°гҖӮеҰӮжһңдёәиҙҹж•°пјҢеҲҷе°ҶиҜҘж•°зҡ„йқһж ҮеәҰеҖјд№ҳд»Ҙ 10 зҡ„иҙҹscale ж¬Ўе№ӮгҖӮ


еҹәжң¬з”Ёжі•

еңЁдҪҝз”ЁBigDecimalзұ»жқҘиҝӣиЎҢи®Ўз®—зҡ„ж—¶еҖҷпјҢдё»иҰҒеҲҶдёәд»ҘдёӢжӯҘйӘӨпјҡ

  1. з”ЁfloatжҲ–иҖ…doubleеҸҳйҮҸжһ„е»әBigDecimalеҜ№иұЎгҖӮ
  2. йҖҡиҝҮи°ғз”ЁBigDecimalзҡ„еҠ пјҢеҮҸпјҢд№ҳпјҢйҷӨзӯүзӣёеә”зҡ„ж–№жі•иҝӣиЎҢз®—жңҜиҝҗз®—гҖӮ
  3. жҠҠBigDecimalеҜ№иұЎиҪ¬жҚўжҲҗfloatпјҢdoubleпјҢintзӯүзұ»еһӢгҖӮ


BigDecimalзұ»жҳҜеҜ№иұЎпјҢдёҚиғҪдҪҝз”Ёдј з»ҹзҡ„+гҖҒ-гҖҒ*гҖҒ/зӯүз®—жңҜиҝҗз®—з¬ҰзӣҙжҺҘеҜ№е…¶иҝӣиЎҢж•°еӯҰиҝҗз®—,иҖҢеҝ…йЎ»и°ғз”Ёе…¶еҜ№еә”зҡ„ж–№жі•.ж–№жі•зҡ„еҸӮж•°д№ҹеҝ…йЎ»жҳҜBigDecimalзұ»еһӢзҡ„еҜ№иұЎгҖӮ

йҰ–е…ҲпјҢдҪҝз”ЁBigDecimalзҡ„жһ„йҖ ж–№жі•жҲ–иҖ…йқҷжҖҒж–№жі•зҡ„valueOf()ж–№жі•жҠҠеҹәжң¬зұ»еһӢзҡ„еҸҳйҮҸжһ„е»әжҲҗBigDecimalеҜ№иұЎ

з„¶еҗҺпјҢеҜ№ж•°жҚ®иҝӣиЎҢеҠ еҮҸд№ҳйҷӨпјҢBigDecimalйғҪжҳҜдёҚеҸҜеҸҳзҡ„пјҲimmutableпјүзҡ„пјҢеңЁиҝӣиЎҢжҜҸдёҖжӯҘиҝҗз®—ж—¶пјҢйғҪдјҡдә§з”ҹдёҖдёӘж–°зҡ„еҜ№иұЎгҖӮ

жіЁж„ҸпјҢеңЁдҪҝз”ЁйҷӨжі•ж—¶пјҢдёҖиҲ¬йңҖиҰҒжҢҮе®ҡзІҫеәҰд»ҘеҸҠиҲҚе…ҘжЁЎејҸпјҢеҰӮжһңдёҚжҢҮе®ҡпјҢдёҖж—ҰеҮәзҺ°ж— йҷҗе°Ҹж•°е°ұдјҡзӣҙжҺҘжҠӣз®—жңҜејӮеёёгҖӮ

жңҖеҗҺпјҢе°Ҷз»“жһңиҪ¬жҚўдёәfloatпјҢdoubleпјҢintзӯүзұ»еһӢ


8з§ҚиҲҚе…ҘжЁЎејҸ

еңЁиҝҗз®—жҲ–иҖ…иҪ¬жҚўзҡ„иҝҮзЁӢдёӯпјҢеҸҜд»Ҙи®ҫе®ҡзІҫеәҰпјҢеҰӮжһңдёҚжҢҮе®ҡиҲҚе…ҘжЁЎејҸпјҢеҲҷй»ҳи®ӨROUND_UNNECESSARYпјҢдҪҶжҳҜиҝҷз§ҚжЁЎејҸж–ӯиЁҖиҜ·жұӮзҡ„ж“ҚдҪңе…·жңүзІҫзЎ®зҡ„з»“жһңпјҢеӣ жӯӨдёҚйңҖиҰҒи®ҫе®ҡзІҫеәҰпјҢжүҖд»ҘеңЁиҝҷдёӘжЁЎејҸдёӢеҺ»и®ҫе®ҡзІҫеәҰпјҢдёҖж—ҰеҮәзҺ°зІҫеәҰжҚҹеӨұпјҢеҝ…з„¶дјҡжҠӣеҮәејӮеёёпјҢзңҹжҳҜеқ‘~~

жүҖд»ҘпјҢи®ҫе®ҡзІҫеәҰж—¶пјҢжңҖеҘҪжҢҮе®ҡиҲҚе…ҘжЁЎејҸгҖӮ


з”ұдәҺBigDecimal дёҚеҸҜеҸҳпјҢжүҖжңүзҡ„еҸҳеҠЁйғҪдјҡиҝ”еӣһж–°зҡ„BigDecimalеҜ№иұЎпјҢжүҖд»ҘйңҖиҰҒдҝқеӯҳж–°еҜ№иұЎжқҘдҪҝз”ЁгҖӮ


иҲҚе…ҘжЁЎејҸдёҖе…ұжңү8з§ҚпјҢйғҪеңЁ BigDecimal и®ҫе®ҡдәҶпјҢе№¶дё”йҖҡиҝҮ RoundingMode иҝҷдёӘжһҡдёҫзұ»иҝӣиЎҢдәҶеҢ…иЈ…гҖӮ

жЁЎејҸ жҸҸиҝ°
ROUND_UP иҲҚе…ҘиҝңзҰ»йӣ¶зҡ„иҲҚе…ҘжЁЎејҸгҖӮеңЁдёўејғйқһйӣ¶йғЁеҲҶд№ӢеүҚе§Ӣз»ҲеўһеҠ ж•°еӯ—(е§Ӣз»ҲеҜ№йқһйӣ¶иҲҚејғйғЁеҲҶеүҚйқўзҡ„ж•°еӯ—еҠ 1)гҖӮжіЁж„ҸпјҢжӯӨиҲҚе…ҘжЁЎејҸе§Ӣз»ҲдёҚдјҡеҮҸе°‘и®Ўз®—еҖјзҡ„еӨ§е°ҸгҖӮ
ROUND_DOWN жҺҘиҝ‘йӣ¶зҡ„иҲҚе…ҘжЁЎејҸгҖӮеңЁдёўејғжҹҗйғЁеҲҶд№ӢеүҚе§Ӣз»ҲдёҚеўһеҠ ж•°еӯ—(д»ҺдёҚеҜ№иҲҚејғйғЁеҲҶеүҚйқўзҡ„ж•°еӯ—еҠ 1пјҢеҚіжҲӘзҹӯ)гҖӮжіЁж„ҸпјҢжӯӨиҲҚе…ҘжЁЎејҸе§Ӣз»ҲдёҚдјҡеўһеҠ и®Ўз®—еҖјзҡ„еӨ§е°ҸгҖӮ
ROUND_CEILING жҺҘиҝ‘жӯЈж— з©·еӨ§зҡ„иҲҚе…ҘжЁЎејҸгҖӮеҰӮжһң BigDecimal дёәжӯЈпјҢеҲҷиҲҚе…ҘиЎҢдёәдёҺ ROUND_UP зӣёеҗҢ;еҰӮжһңдёәиҙҹпјҢеҲҷиҲҚе…ҘиЎҢдёәдёҺ ROUND_DOWN зӣёеҗҢгҖӮжіЁж„ҸпјҢжӯӨиҲҚе…ҘжЁЎејҸе§Ӣз»ҲдёҚдјҡеҮҸе°‘и®Ўз®—еҖјгҖӮ
ROUND_FLOOR жҺҘиҝ‘иҙҹж— з©·еӨ§зҡ„иҲҚе…ҘжЁЎејҸгҖӮеҰӮжһң BigDecimal дёәжӯЈпјҢеҲҷиҲҚе…ҘиЎҢдёәдёҺ ROUND_DOWN зӣёеҗҢ;еҰӮжһңдёәиҙҹпјҢеҲҷиҲҚе…ҘиЎҢдёәдёҺ ROUND_UP зӣёеҗҢгҖӮжіЁж„ҸпјҢжӯӨиҲҚе…ҘжЁЎејҸе§Ӣз»ҲдёҚдјҡеўһеҠ и®Ўз®—еҖјгҖӮ
ROUND_HALF_UP еҗ‘вҖңжңҖжҺҘиҝ‘зҡ„вҖқж•°еӯ—иҲҚе…ҘпјҢеҰӮжһңдёҺдёӨдёӘзӣёйӮ»ж•°еӯ—зҡ„и·қзҰ»зӣёзӯүпјҢеҲҷдёәеҗ‘дёҠиҲҚе…Ҙзҡ„иҲҚе…ҘжЁЎејҸгҖӮеҰӮжһңиҲҚејғйғЁеҲҶ >= 0.5пјҢеҲҷиҲҚе…ҘиЎҢдёәдёҺ ROUND_UP зӣёеҗҢ;еҗҰеҲҷиҲҚе…ҘиЎҢдёәдёҺ ROUND_DOWN зӣёеҗҢгҖӮжіЁж„ҸпјҢиҝҷжҳҜжҲ‘д»¬еӨ§еӨҡж•°дәәеңЁе°ҸеӯҰж—¶е°ұеӯҰиҝҮзҡ„иҲҚе…ҘжЁЎејҸ(еӣӣиҲҚдә”е…Ҙ)гҖӮ
ROUND_HALF_DOWN еҗ‘вҖңжңҖжҺҘиҝ‘зҡ„вҖқж•°еӯ—иҲҚе…ҘпјҢеҰӮжһңдёҺдёӨдёӘзӣёйӮ»ж•°еӯ—зҡ„и·қзҰ»зӣёзӯүпјҢеҲҷдёәдёҠиҲҚе…Ҙзҡ„иҲҚе…ҘжЁЎејҸгҖӮеҰӮжһңиҲҚејғйғЁеҲҶ > 0.5пјҢеҲҷиҲҚе…ҘиЎҢдёәдёҺ ROUND_UP зӣёеҗҢ;еҗҰеҲҷиҲҚе…ҘиЎҢдёәдёҺ ROUND_DOWN зӣёеҗҢ(дә”иҲҚе…ӯе…Ҙ)гҖӮ
ROUND_HALF_EVEN еҗ‘вҖңжңҖжҺҘиҝ‘зҡ„вҖқж•°еӯ—иҲҚе…ҘпјҢеҰӮжһңдёҺдёӨдёӘзӣёйӮ»ж•°еӯ—зҡ„и·қзҰ»зӣёзӯүпјҢеҲҷеҗ‘зӣёйӮ»зҡ„еҒ¶ж•°иҲҚе…ҘгҖӮеҰӮжһңиҲҚејғйғЁеҲҶе·Ұиҫ№зҡ„ж•°еӯ—дёәеҘҮж•°пјҢеҲҷиҲҚе…ҘиЎҢдёәдёҺ ROUND_HALF_UP зӣёеҗҢ;еҰӮжһңдёәеҒ¶ж•°пјҢеҲҷиҲҚе…ҘиЎҢдёәдёҺ ROUND_HALF_DOWN зӣёеҗҢгҖӮжіЁж„ҸпјҢеңЁйҮҚеӨҚиҝӣиЎҢдёҖзі»еҲ—и®Ўз®—ж—¶пјҢжӯӨиҲҚе…ҘжЁЎејҸеҸҜд»Ҙе°ҶзҙҜеҠ й”ҷиҜҜеҮҸеҲ°жңҖе°ҸгҖӮжӯӨиҲҚе…ҘжЁЎејҸд№ҹз§°дёәвҖңй“¶иЎҢе®¶иҲҚе…Ҙжі•вҖқпјҢдё»иҰҒеңЁзҫҺеӣҪдҪҝз”ЁгҖӮеӣӣиҲҚе…ӯе…ҘпјҢдә”еҲҶдёӨз§Қжғ…еҶөгҖӮеҰӮжһңеүҚдёҖдҪҚдёәеҘҮж•°пјҢеҲҷе…ҘдҪҚпјҢеҗҰеҲҷиҲҚеҺ»гҖӮд»ҘдёӢдҫӢеӯҗдёәдҝқз•ҷе°Ҹж•°зӮ№1дҪҚпјҢйӮЈд№Ҳиҝҷз§ҚиҲҚе…Ҙж–№ејҸдёӢзҡ„з»“жһңгҖӮ1.15 -> 1.2 1.25 -> 1.2
ROUND_UNNECESSARY й»ҳи®Өзҡ„жЁЎејҸпјҢж–ӯиЁҖиҜ·жұӮзҡ„ж“ҚдҪңе…·жңүзІҫзЎ®зҡ„з»“жһңпјҢеӣ жӯӨдёҚйңҖиҰҒиҲҚе…ҘгҖӮеҰӮжһңеҜ№иҺ·еҫ—зІҫзЎ®з»“жһңзҡ„ж“ҚдҪңжҢҮе®ҡжӯӨиҲҚе…ҘжЁЎејҸпјҢеҲҷжҠӣеҮәArithmeticExceptionгҖӮ

иҝҷ8з§ҚжЁЎејҸдёӯпјҢжңҖеёёз”Ёзҡ„еә”иҜҘжҳҜ ROUND_HALF_UP еӣӣиҲҚдә”е…ҘпјҢиҝҷдёӘйғҪзҹҘйҒ“гҖӮ

иҝҳжңүдёҖз§ҚROUND_HALF_DOWN дә”иҲҚе…ӯе…ҘпјҢиҝҷдёӘжҳҜжЁЎејҸдёӢпјҢиҰҒзңӢзІҫеәҰзҡ„еҗҺйқўзҡ„ж•°еҖјпјҢдёҖж—Ұи¶…иҝҮ5 пјҢе°ұе…ҘдҪҚгҖӮ

еңЁиҝҷдёӘжЁЎејҸдёӢеҸҜд»ҘзңӢеҲ°пјҢеҚідҪҝжҳҜ3.14150001 пјҢеңЁдҝқз•ҷ3дҪҚе°Ҹж•°зҡ„жғ…еҶөдёӢпјҢз»“жһңдҫқз„¶жҳҜиҝӣдёҖдҪҚгҖӮ


жңҖйҡҫзҗҶи§Јзҡ„жҳҜROUND_HALF_EVEN й“¶иЎҢе®¶иҲҚе…ҘпјҢеӣӣиҲҚе…ӯе…ҘпјҢдә”еҲҶдёӨз§Қжғ…еҶөпјҢеҰӮжһңеүҚдёҖдҪҚдёәеҘҮж•°пјҢеҲҷе…ҘдҪҚпјҢеҗҰеҲҷиҲҚеҺ»гҖӮиҝҷз§ҚжЁЎејҸеҸҜд»Ҙе°Ҷй”ҷиҜҜеҮҸеҲ°жңҖе°ҸпјҢжүҖд»ҘдёҖиҲ¬еңЁй“¶иЎҢдҪҝз”ЁгҖӮ

зңӢеҲ°иҝҷдёӘзӨәдҫӢпјҢеә”иҜҘеҫҲжё…жҘҡзҡ„дәҶи§Јиҝҷз§ҚжЁЎејҸгҖӮ


еӣӣиҲҚдә”е…Ҙеёёз”ЁеҶҷжі•

дёҖгҖҒMathпјҲе°Ҹж•°иҲҚе…Ҙдёәж•ҙж•°пјү

Math.ceil() еҗ‘дёҠиҲҚе…ҘпјҢеҚіе®ғжҖ»жҳҜе°Ҷж•°еҖјеҗ‘дёҠиҲҚе…ҘдёәжңҖжҺҘиҝ‘зҡ„ж•ҙж•°пјӣ

Math.floor() еҗ‘дёӢиҲҚе…ҘпјҢеҚіе®ғжҖ»жҳҜе°Ҷж•°еҖјеҗ‘дёӢиҲҚе…ҘдёәжңҖжҺҘиҝ‘зҡ„ж•ҙж•°пјӣ

Math.round()ж ҮеҮҶиҲҚе…ҘпјҢеҚіе®ғжҖ»жҳҜе°Ҷж•°еҖјеӣӣиҲҚдә”е…ҘдёәжңҖжҺҘиҝ‘зҡ„ж•ҙж•°гҖӮ


дәҢгҖҒBigDecimal

иҝҷз§Қж–№ејҸе°ұжҳҜеүҚйқўиҜҙзҡ„и®ҫзҪ®зІҫеәҰе’ҢиҲҚе…ҘжЁЎејҸгҖӮ

иҝҷйҮҢе°ұдёҚеӨҡиҜҙдәҶгҖӮ


дёүгҖҒDecimalFormat

DecimalFormat й»ҳи®ӨйҮҮз”ЁдәҶRoundingMode.HALF_EVENиҝҷз§ҚиҲҚе…ҘжЁЎејҸпјҢеҸҜд»ҘйҖҡиҝҮ setRoundingMode() жқҘйҮҚж–°и®ҫзҪ®иҲҚе…ҘжЁЎејҸпјҢиҖҢдё”format д№ӢеҗҺзҡ„з»“жһңжҳҜдёҖдёӘеӯ—з¬ҰдёІзұ»еһӢгҖӮ

DecimalFormat зұ»дё»иҰҒйқ  # е’Ң 0 дёӨз§ҚеҚ дҪҚз¬ҰеҸ·жқҘжҢҮе®ҡж•°еӯ—й•ҝеәҰгҖӮ0 иЎЁзӨәеҰӮжһңдҪҚж•°дёҚи¶іеҲҷд»Ҙ 0 еЎ«е……пјҢ# иЎЁзӨәеҸӘиҰҒжңүеҸҜиғҪе°ұжҠҠж•°еӯ—жӢүдёҠиҝҷдёӘдҪҚзҪ®гҖӮ
